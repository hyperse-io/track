# Installation

## Requirements

- [Node.js](https://nodejs.org/en/) v18 or above, with support for even-numbered Node.js versions.

## Install

```bash
npm install @hyperse/track --save
```

or you can use **yarn**

```bash
yarn add @hyperse/track
```

We have completed installing the package.
