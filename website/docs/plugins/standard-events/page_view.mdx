<NpmLink name="common" />

This event indicates that the user viewed the web page.

## Parameters

### page_location

<Property /> `string`

The URL of the page.

### page_path

<Property /> `string`

The path of the page.

### client_id

<Property /> `string`

The client ID.

### language

<Property /> `string`

The language used by the client. See language tags and codes. [LANGUAGE](https://en.wikipedia.org/wiki/Language_localisation#Language_tags_and_codes)

### page_encoding

<Property /> `string`

The encoding of the page.

### page_title

<Property /> `string`

The title of the page.

### user_agent

<Property /> `string`

The user agent of the client
